## Medical Density

The medical density problem is now mostly a placement problem, not a global problem.

The archive still contains EGF, steroid conjugates, HPG axis, transdermal absorption, pharmaceutical analogies, clinical critique, institutional analysis, and protocol language. Those terms are not gone, and they should not all be gone. They belong in analysis rooms, synthesis notes, Dictionary entries, and method pages.

What changed is that they no longer control the first nervous-system experience of the reader.

Remaining density to watch:

- The `README.md` index still necessarily lists analysis files with dense descriptions. This is acceptable because it is an index, but the reader path above it must remain primary.
- `synthesis-current-state-may-2026.md` is still somewhat research-forward in places. It works as a state report, but if it remains the endpoint of the main path for long, it will eventually want a June or post-clearing successor rather than endless patching.
- The suppression/lineage pieces can still become argument-heavy. They are powerful, but they should remain rooms off the path unless a specific piece is doing threshold work.
- Method pages with `protocol` in the title remain sensitive as public-facing material. Their position as reference rooms helps, but they should keep strong non-instructional framing.

The present balance is good. The rule going forward should be simple: the path carries experience; the rooms carry density.
